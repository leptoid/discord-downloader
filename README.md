# Discord Downloader

Automatically download files uploaded to Discord.

Usage:

```
discord-downloader your_email your_password
```

Files will be saved in the current folder using following structure:

```
<Current Folder>
│
├── Direct Messages
│   │
│   ├── User 1
│   │   ├── file1.jpg
│   │   └── file2.png
│   │
│   └── User 2
│       ├── file1.jpg
│       └── file2.png
│
├── Server 1
│   │
│   ├── channel1
│   │   ├── file1.jpg
│   │   └── file2.png
│   │
│   └── channel2
│       ├── file1.jpg
│       └── file2.png
│
└── Server 2
    │
    ├── channel1
    │   ├── file1.jpg
    │   └── file2.png
    │
    └── channel2
        ├── file1.jpg
        └── file2.png
```
